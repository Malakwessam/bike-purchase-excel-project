# bike-purchase-excel-project
This project explores a bike sales dataset to understand customer purchasing behavior

Overview
This project analyzes a Bike Sales dataset to understand customer purchasing behavior and identify factors influencing bike purchases. The analysis was conducted using Microsoft Excel and visualized through an interactive dashboard.

Dataset
The dataset contains demographic and behavioral information about customers, including income, age group, occupation, and commute distance.

Process
Data cleaning and preprocessing
Data exploration using Pivot Tables
Creation of visualizations to highlight purchasing patterns
Development of a dashboard for interactive analysis

Key Insights
Relationship between income level and bike purchases
Bike purchases across different age groups
Impact of commute distance on purchasing behavior
Customer demographic trends

Tools Used
Microsoft Excel
Pivot Tables
Data Visualization (Charts)
